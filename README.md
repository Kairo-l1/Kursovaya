# 🛣️ Система детекции дорожных дефектов (YOLOv8)

## Описание

Веб-приложение на Gradio для автоматического обнаружения и классификации дефектов дорожного покрытия с помощью модели YOLOv8.

**Основные возможности:**
- Загрузка изображения через интерфейс
- Автоматическое обнаружение дефектов (5 классов)
- Аннотированное изображение с цветными боксами
- Текстовое резюме с таблицей результатов
- JSON-вывод для программной обработки
- Регулируемый порог уверенности (slider 0.05–0.5)

## Требования

- Python 3.10+
- Виртуальное окружение (рекомендуется)

## Установка локально

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/USERNAME/REPO.git
cd REPO
```

2. **Создайте виртуальное окружение:**
```bash
python -m venv .venv

# Windows
.\.venv\Scripts\activate

# macOS / Linux
source .venv/bin/activate
```

3. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

Готово! Модель уже включена в проект.

## Запуск приложения

```bash
python app.py
```

Откройте в браузере: **http://127.0.0.1:7861**

## Файл .gitignore

Репозиторий включает файл `.gitignore`, который исключает:
- Результаты тренировки (`runs/`, `eval_results/`, `debug_output/`)
- Виртуальное окружение (`.venv/`)
- IDE конфигурацию (`.idea/`, `.vscode/`)
- Кэш Python (`__pycache__/`)
- Временные файлы и логи

**Модель `best.pt` (~6 МБ) включена в репозиторий,** так как это необходимо для работы приложения после клонирования.
